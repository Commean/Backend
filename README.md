# Backend
Backend for the Commean project.

## Compile

Use appropriate scripts based on the OS:

### Linux
```sh
scripts/build.sh
```

### Windows
```powershell
scripts/build.ps1
```


## Start
The database has to be started BEFORE the Backend.


### Linux
```sh
scripts/startDB.sh
scripts/run.sh
```

### Windows
```powershell
scripts/startDB.ps1
scripts/run.ps1
```
